1. Introdução

Objetivo: Facilitar o cadastro, consulta, edição e exclusão de pessoas e produtos de forma simples e intuitiva.

Público-Alvo: Usuários leigos e intermediários, sem necessidade de conhecimentos técnicos avançados.

Visão Geral: O sistema possui abas para cadastro e visualização, permitindo operações básicas em pessoas e produtos.

2. Acesso à Aplicação

  • Não há sistema de login, o acesso é direto pela URL da aplicação.
  • Recomendado utilizar navegadores atualizados como Chrome, Firefox, Edge ou Safari.

3. Cadastro de Pessoas

  1. Acesse a aba Cadastro de Pessoas.
  2. Preencha os campos:
    • Nome: Informe o nome completo.
    • Idade: Informe a idade (número inteiro).
  3. Clique no botão Cadastrar.
  4. O ID será gerado automaticamente.
  5. A pessoa aparecerá na lista da aba de visualização.

4. Cadastro de Produtos

  1. Acesse a aba Cadastro de Produtos.
  2. Preencha os campos:
    • Nome: Nome do produto.
    • Código: Código único do produto.
    • Quantidade: Quantidade disponível.
    • Preço: Preço unitário (exemplo: 15.99).
  3. Clique no botão Cadastrar.
  4. O produto será listado na aba de visualização.

5. Listagem, Edição e Exclusão

Pessoas

  • Visualize todas as pessoas cadastradas na aba correspondente.
  • Alterar Idade: Clique no botão Alterar Idade ao lado da pessoa desejada, insira a nova idade e confirme.
  • Excluir: Clique no botão Excluir para remover a pessoa do sistema. Confirme a ação.

Produtos

  • Visualize todos os produtos cadastrados na aba correspondente.
  • Editar: Clique no botão Editar, modifique os dados e salve as alterações.
  • Excluir: Clique no botão Excluir para remover o produto. Confirme a ação.

6. Solução de Problemas Comuns

  • Campos obrigatórios: Preencha todos os campos para evitar erros ao cadastrar.
  • Cadastro duplicado: Certifique-se de que os códigos de produtos são únicos.
  • Atualização da lista: Atualize a página para visualizar as mudanças após cadastro, edição ou exclusão.
  • Erro de conexão: Verifique sua conexão com a internet ou o servidor do banco de dados.
